Custode Builder

The marketing funnel (hero → how it works → prompt to create → preview paywall) is served by PHP at the site root, not this static file.

From the project root, run:

php -S 127.0.0.1:8080 index.php

Then open http://127.0.0.1:8080/ for the landing page, http://127.0.0.1:8080/start for the creation wizard, and http://127.0.0.1:8080/admin for admin sign-in.
